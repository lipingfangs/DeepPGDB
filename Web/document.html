DeepPGDB Bioinformatics Assistant

South China Agricultural University Genomics Bioinformatics Tool

Introduction

The SCAU Genomics Bioinformatics Assistant is a specialized tool developed by South China Agricultural University to automate common bioinformatics tasks with simple commands. It follows strict rules to generate appropriate commands based on user input.

Core Functions

1. Sequence Analysis

Example inputs for sequence comparison:

Input Example 1: 请将序列GCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCG比对到拟南芥数据库。 Input Example 2: Please compare sequence MKTIIALSYIFCLVFADYKDDDDAAAGGGGGLAALWLLPLALALPLALWLAQKAAQAAAGAAA to rice database.
2. Genome Data Retrieval

Example inputs for genome operations:

Input Example 1: 请下载水稻基因组。 Input Example 2: Download the tomato genome. Input Example 3: 请帮我提取水稻基因组3号染色体上322112至332112位置间的序列。 Input Example 4: Extract sequence from position 1000000 to 2000000 on chromosome 5 of Arabidopsis.
3. Gene Information Search

Example inputs for gene searches:

Input Example 1: 请帮搜索水稻的MYB转录因子相关的基因。 Input Example 2: Search for NAC transcription factors in rice. Input Example 3: 帮我严格搜索AP2相关的基因在拟南芥中。 Input Example 4: Strictly search for WRKY genes in tomato.
4. Expression Analysis

Example inputs for expression data:

Input Example 1: 请帮我查找到Os11g0147600基因在水稻中的表达量, 并绘图。 Input Example 2: Find expression of gene AT1G01090 in Arabidopsis and plot. Input Example 3: 请查找到AT1G01090基因在拟南芥中的表达量。
5. Variation Analysis

Example inputs for variation data:

Input Example 1: 请帮我提取水稻群体1号染色体上122112至125112位置间的变异位点, 并绘图。 Input Example 2: Extract variants from position 500000 to 600000 on chromosome 2 of bitter gourd.
6. Enrichment Analysis

Example inputs for enrichment analysis:

Input Example 1: 请将拟南芥基因AT5G22060.1,AT1G03000.1,AT2G44790.1进行富集分析,并绘图。 Input Example 2: Perform GO enrichment analysis for rice genes Os11t0147600-01,Os02t0802500-01.
7. Sequence Extraction

Example inputs for sequence extraction:

Input Example 1: 请帮我提取水稻基因Os12g0541500的序列。 Input Example 2: Extract sequence of gene AT5G16970.1 from Arabidopsis. Input Example 3: 请帮我提取拟南芥AT5G22060基因的序列。

Species Naming Convention

The tool uses specific Latin binomial names for organisms:

Common Name Latin Binomial
Arabidopsis Arabidopsis.thaliana
Rice Oryza.sativa
Tomato Solanum.lycopersicum
Bitter gourd Momordica.charantia
Guava Psidium.guajava
Apple Malus.domestica
Watermelon Citrullus.lanatus
Sweet orange Citrus.sinensis
Note: Species names must be used exactly as shown, with no changes to capitalization or formatting.

File Naming Convention

All genomic files follow a strict naming pattern based on Latin binomial names:

  • Genome file: Latin_binomial.genome (e.g., Arabidopsis.thaliana.genome)
  • Annotation file: Latin_binomial.genome.ann
  • Expression file: Latin_binomial.exp.csv
  • Gene file: Latin_binomial.genome.gene
  • GTF file: Latin_binomial.genome.gtf
